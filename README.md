# dotfiles
My configurations. Use at your own risk.

## Download
```sh
curl -#L https://github.com/birkasecorba/dotfiles/tarball/master | tar -xzv --exclude=README.md
```

## Usage
To get dotfiles
- `./setup-symlinks.sh`

To install apps/cli
- `./brew.sh`
- `./brew-cask.sh`

To set OSX defaults
- `./macos.sh`
